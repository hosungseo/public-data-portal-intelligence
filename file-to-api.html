경량 공개 뷰
브라우저는 전체 내부 마스터가 아니라 요약 자산만 읽으므로, 페이지는 빠르고 읽기 쉬운 검토 화면에 집중합니다.

수요 우선

교차수요 확인
파일 수요와 API형 수요가 함께 관찰되어, 더 정돈된 제공방식에 대한 압력이 높은 후보입니다.
